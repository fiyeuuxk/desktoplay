这是一个java写的桌面背景播放软件
使用的是mpv播放器，这个是我目前用过的性能最强的播放器，体积小，官网https://mpv.io/
C:\Users\Administrator\Documents\02-软件\05-动画\mpv
这是我的mpv目录
你要使用这个软件要去官网下载mpv播放器放到对应目录或自己修改代码更改路径
还要安装JDK25官网https://jdk.java.net/25/
在mpv文件夹中新建一个portable_config文件夹
在portable_config文件夹中新建一个mpv.conf

文件内容是loop-playlist=inf这样就能循环播放视频了
